# LeetCode  96

> [96. 不同的二叉搜索树 - 力扣（LeetCode）](https://leetcode.cn/problems/unique-binary-search-trees/description/)



## summary

> 动态规划可以从递归入手  但是递归有缺陷就是一些数据可能会重复计算
>
> 基于这样我们可以挂一个缓存表  就有了所谓的记忆化搜索 这个记忆化搜索其实就是自顶至底的DP
>
> 但是严格的DP是自底至顶的 因此又改出一个严格DP



> 给你一个整数 `n` ，求恰由 `n` 个节点组成且节点值从 `1` 到 `n` 互不相同的 **二叉搜索树** 有多少种？返回满足题意的二叉搜索树的种数。

> 分析：这是一个二叉搜索树  中序遍历序列是有序的  **问题可以转化为在[1,n]有多少种建树方式**
>
> 这个问题其实我在学数据结构的时候知道这是一个卡特兰数  但是我通过阶乘的方法没有全部通过 



## code

> 从递归入手动态规划：
>
> > ```Java
> > public int numTrees(int n) {
> >         if(n==0||n==1){
> >             //在空树或者只有一个节点的时候只存在唯一建树方案
> >             return 1;
> >         }
> >         int num=0;
> >         for(int i=1;i<=n;i++){
> >             //相当于把数组下标为i位置的树作为根节点 然后[1,i-1]做左子树 [i+1,n]做右子树组合起来
> >             //左边i-1个节点 右边n-i个节点
> >             num+=numTrees(i-1)*numTrees(n-i);
> >         }
> >         return num;
> >     }
> > ```
>
>  
>
> 记忆化搜索：
>
> > ```Java
> > public int numTrees(int n){
> >         int[] dp=new int[n+1];
> >         Arrays.fill(dp,-1);
> >         return f2(n,dp);
> >     }
> > 
> >     public int f2(int n,int[] dp){
> >         if(n==0||n==1){
> >             return 1;
> >         }
> >         if(dp[n]!=-1){
> >             return dp[n];
> >         }
> > 
> >         int num=0;
> >         for(int i=1;i<=n;i++){
> >             num+=f2(i-1,dp)*f2(n-i,dp);
> >         }
> >         dp[n]=num;
> >         return num;
> >     }
> > ```
>
>  
>
> 自下而上严格DP:
>
> ```Java
> public int f3(int n){
>         if(n==0||n==1){
>             return 1;
>         }
>         int[] dp=new int[n+1];
>         dp[0]=1;
>         for(int i=1;i<=n;i++){
>             for(int j=0;j<i;j++){
>                 //这里i表示总的个数
>                 //j表示左子树分的个数  n-j-1则表示右子树分的个数
>                 //左子树最多i-1个
>                 dp[i]+=dp[j]*dp[i-j-1];
>             }
>         }
> 
>         return dp[n];
>     }
> ```
>
>  
>
> 显然，卡特兰数需要用到C(0)~C(n-1)  因此用DP方法不能再优化空间复杂度了
>
> 但是可以用数学方法来实现卡特兰数的计算  这个通过递推去实现比我之前直接求阶乘要好
>
> 它运算更快O(n) 而直接算阶乘O(3n) 还能避免大数溢出
>
> ```text
> 卡特兰数公式：C(2n, n) / (n + 1)
> 推导一下就有：
> C(n) / C(n-1) = [C(2n, n)/(n+1)] / [C(2n-2, n-1)/n]
>               = (4n - 2) / (n + 1)
> ```
>
> 
>
> ```Java
> public int numTrees(int n){
>         //卡特兰数公式：C(2n, n) / (n + 1)
>         long ans=1;//相当于C(0)=1  循环n次计算C(n)
>         for(int i=1;i<=n;i++){
>             ans=ans*2*(2*i-1)/(i+1);
>         }
>         return (int)ans;
>     }
> ```
>
